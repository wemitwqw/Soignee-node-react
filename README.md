* Sign-up & sign-in
* Admin view 
* Follow a user

## How to use
Navigate to docker

Run:
* docker-compose run --rm --no-deps backend-node install
* docker-compose run --rm --no-deps frontend-react install
* docker-compose up -d

To stop run:
* docker-compose stop
